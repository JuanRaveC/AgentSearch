Nota de Resumen
El libro destaca tres aspectos importantes de las estructuras de datos. Primero, se hace hincapie en la conexion que hay entre las estructuras de datos y sus algoritmos. segundo, las estructuras se presentan con orientación a objetos. Finalmente, se le da importancia a la implementación, lo cual lleva a los autores a escoger Java como lenguaje de programacion para hacerla.
